# tcc-front-end-2


Shell abaixo é necessário para rodar o servidor front end.
```sh
    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.4/install.sh | bash
    source ~/.bashrc
    $ nvm --version
    $ nvm install 18
    $ nvm use 18
    $ node -v
    $ rm rf node_modules/
    $ sudo rm -rf node_modules
    $ git status
    $ git restore .
    $ npm install
```
